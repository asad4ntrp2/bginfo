# Changelog

All notable changes to BGInfo for Linux will be documented in this file.

## [1.0.0] - 2026-02-14

### Added
- Initial release
- One-command installer for all major Linux distributions
- Auto-detection of CPU, RAM, network interface, disk, hostname
- Dynamic CPU core display (adapts to any core count)
- Dark transparent theme with GitHub-inspired color scheme
- Live network speed graphs
- Top processes by CPU and memory
- Service monitoring (SSH, Docker, Firewall)
- Desktop autostart configuration
- Wayland to X11 auto-switching for compatibility
- Uninstall support via --uninstall flag
- Support for apt, dnf, yum, pacman, zypper package managers
